Punk API Collector

This website, collects data from the Punk API

Once the data is collected, it shows it in a good visual way so you can choose your next beer!

How it works?

In All Beers section you'll find a list of all the beers in the API. You can check a specifc beer by clicking on its image.

Also, if you scroll all the way down, you'll be able to load more beers.

In Random section you will get a random beer from the API.

Note: Please, bear in mind, the locations in the maps are not related to the actual place where the beer is being made. Is shown just as a learning purpose
